# 🧭 DevCompass

> **The ultimate interactive guidebook and AI-powered consultant for aspiring developers.**

DevCompass is a modern web application built with React and Tailwind CSS designed to help new programmers navigate the overwhelming world of tech. Whether you are looking for the right programming language, the industry-standard tools for your specific domain, or personalized project advice from an AI mentor, DevCompass has you covered.

## ✨ Features

DevCompass is split into three highly interactive sections:

* **🌐 The Language Pyramid:** A massive, searchable directory of programming languages categorized by their abstraction level and primary use case. 
* **🛠️ Development Tools Catalog:** A curated, filterable database of frameworks, IDEs, databases, and DevOps tools sorted by development domains (Web, Mobile, Game Dev, AI, etc.).
* **🤖 AI Project Advisor:** A built-in, context-aware AI chatbot powered by the Gemini 2.5 Flash API. Describe a project you want to build, and the AI will recommend the exact tech stack you need, formatted beautifully in Markdown.

## 🚀 Tech Stack

This project was built focusing on performance, modern UI/UX (featuring a clean, neo-brutalist aesthetic), and seamless API integration.

* **Frontend:** React 18, Vite
* **Styling:** Tailwind CSS
* **Icons:** `react-icons`, `lucide-react`
* **AI Integration:** `@google/genai` (Gemini API)
* **Markdown Parsing:** `react-markdown`
* **State Management:** React Hooks (`useState`, `useEffect`) & Browser `sessionStorage`
